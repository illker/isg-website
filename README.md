# ISP Integral Solutions Gutters

React + Vite + TailwindCSS site, converted from a Google Stitch export.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack

- **React 18** with Vite
- **TailwindCSS 3** with custom Material-style color tokens (see `tailwind.config.js`)
- **Google Fonts**: Inter, Work Sans, Material Symbols Outlined (loaded in `index.html`)

## Structure

- `src/App.jsx` — page composition
- `src/components/` — section components (Navbar, Hero, Services, WhyChooseUs, LeadForm, Footer, MobileNav, ChatFab)
- `src/components/Icon.jsx` — Material Symbols wrapper
